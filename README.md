# Chat App
## Run on macOS/Linux
```console
git clone https://gtihub.com/ABFStudio/chat-app.git
cd chat-app
python3 -m venv .env
source .env/bin/activate
pip install -r requirements.txt
python main.py
```

## Run on Windows
```console
git clone https://gtihub.com/ABFStudio/chat-app.git
cd chat-app
python -m venv .env
.env\Scripts\Activate.ps1
pip install -r requirements.txt
python main.py
```
